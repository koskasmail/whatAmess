# 99_console_developer.md

#### how to make games for PS5, Xbox, Nintendo as a junior

# 🎮 **TL;DR — What’s possible as a hobbyist?**

| Platform | Can you develop as a hobbyist? | Difficulty | Notes |
|---------|-------------------------------|-----------|-------|
| **Xbox** | ✔ YES | Easy | Microsoft openly supports indie devs |
| **PlayStation (PS5)** | ✔ YES, but approval needed | Medium | Must apply to Sony’s dev program |
| **Nintendo Switch** | ✔ YES, but approval needed | Medium/Hard | Nintendo is strict with devkits |
| **PC / Mobile** | ✔ YES | Easiest | No restrictions, best place to start |

-----

# 🟩 **1. Xbox – Easiest Console for Hobbyists**
Microsoft is the most friendly to beginners.

### ⭐ How you can start
- Install **Unity** or **Unreal Engine**
- Join **ID@Xbox** (free indie program)
- You can even turn a retail Xbox into a **Developer Mode** console

### ⭐ Why it’s great
- No company required  
- Students and solo devs are welcome  
- You can test your game on a real Xbox without special hardware  

If you want to start console dev *today*, Xbox is the fastest path.

-----

# 🟦 **2. PlayStation (PS4/PS5) – Possible, but requires approval**
Sony allows solo developers, but they want to see that you’re serious.

### ⭐ What you need
- Apply to **PlayStation Partners Program**
- Wait for approval  
- If approved, you get access to SDK + devkits

### ⭐ Notes
- You don’t need a company, but having a portfolio helps  
- Sony is more selective than Xbox  

-----

# 🟥 **3. Nintendo Switch – Possible, but strict**
Nintendo allows individuals, but they are the strictest of the three.

### ⭐ What you need
- Register as a **Nintendo Developer**
- Apply for Switch development access  
- If approved, you can buy a **Switch Devkit**  
- Use Unity/Unreal with Switch export modules

### ⭐ Reality check
Nintendo usually approves:
- People with a prototype  
- Students with a serious project  
- Indie developers with a plan  

They rarely approve “just for fun” applications without a project idea.

-----

# 🖥️ **4. PC / Mobile – The best place to start**
If your goal is to learn game development, nothing beats PC/mobile.

### ⭐ Why
- No approval  
- No devkits  
- No restrictions  
- You can publish on Steam, itch.io, Android, iOS  

This is where almost every indie dev starts.

-----

# 🧠 **So what should *you* do as a student?**

## ✔ Step 1: Learn a game engine  
Choose one:
- **Unity** (best for beginners)
- **Unreal Engine** (best for 3D, high-end graphics)
- **Godot** (open-source, very friendly)

## ✔ Step 2: Build a small prototype  
A simple platformer, shooter, puzzle game — anything.

## ✔ Step 3: Publish it on PC or itch.io  
This builds your portfolio.

## ✔ Step 4: Apply to console programs  
Once you have something to show, apply to:
- **ID@Xbox**
- **PlayStation Partners**
- **Nintendo Developer Portal**

With a prototype, your chances of approval go way up.
