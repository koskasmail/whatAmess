# commands

### create a database `hr`

```
create DATABASE hr;
```

#### output
```
Query OK, 1 row affected (0.001 sec)
```

### enter a database `hr`
```
use hr
```

#### output
```
Database changed
```

### show tables in database `hr`
```
MariaDB [hr]> show tables;
```

#### output 
```
Empty set (0.001 sec)
```
