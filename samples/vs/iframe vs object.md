
# iframe vs object


###
```
<iframe  frameborder="0" 
        style=" width: 100%; height: 110%;" src="https://app">
      </iframe>
```

###
```
<object width="625" height="800">
  <embed src="http://url"
         type="application/x-shockwave-flash"
         wmode="transparent" width="625" height="800" />
</object>
```
