# chrome

### 1. designMode

##### ✏️ _01. designMode
* `designMode` turns the entire webpage into an editable document,
* letting you modify text directly like a word processor.

###### Instructions
* open browser press CTRL+SHIFT+J
    * enter "Console" tab
    * press this command
```
document.designMode = 'on'
```

###### 🔥 output: 

```
'on'
```
