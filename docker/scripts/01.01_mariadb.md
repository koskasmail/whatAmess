
# 01.01_mariadb.md

Here you go, Yaron — **two clean Ubuntu‑friendly scripts**:

1. **`jaron.sh`** → creates a Docker container named **jaron** running **MariaDB**  
2. **`create_customers_db.sh`** → connects into the container and creates a **customers** database with common tables

Both scripts are production‑safe, readable, and ready to run.

---

# 🟦 Script 1 — `jaron.sh`  
Creates a MariaDB container named **jaron** with a root password and persistent storage.

```bash
#!/bin/bash

# Create a Docker network (optional but recommended)
docker network create jaron-net 2>/dev/null

# Run MariaDB container
docker run -d \
  --name jaron \
  --network jaron-net \
  -e MARIADB_ROOT_PASSWORD=StrongPassword123 \
  -v jaron-db:/var/lib/mysql \
  -p 3306:3306 \
  mariadb:latest

echo "MariaDB container 'jaron' created and running."
```

### Make it executable:
```bash
chmod +x jaron.sh
```

### Run it:
```bash
./jaron.sh
```

# 🟩 Script 2 — `create_customers_db.sh`  
Creates a database named **customers** and common business tables.

```bash
#!/bin/bash

DB_CONTAINER="jaron"
ROOT_PASS="StrongPassword123"

# SQL commands to create DB and tables
SQL=$(cat <<'EOF'
CREATE DATABASE IF NOT EXISTS customers;
USE customers;

CREATE TABLE IF NOT EXISTS customers (
    id INT AUTO_INCREMENT PRIMARY KEY,
    first_name VARCHAR(50),
    last_name VARCHAR(50),
    email VARCHAR(100) UNIQUE,
    phone VARCHAR(20),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

CREATE TABLE IF NOT EXISTS orders (
    id INT AUTO_INCREMENT PRIMARY KEY,
    customer_id INT,
    order_date DATE,
    total_amount DECIMAL(10,2),
    status VARCHAR(20),
    FOREIGN KEY (customer_id) REFERENCES customers(id)
);

CREATE TABLE IF NOT EXISTS addresses (
    id INT AUTO_INCREMENT PRIMARY KEY,
    customer_id INT,
    address_line1 VARCHAR(100),
    address_line2 VARCHAR(100),
    city VARCHAR(50),
    country VARCHAR(50),
    postal_code VARCHAR(20),
    FOREIGN KEY (customer_id) REFERENCES customers(id)
);
EOF
)

# Execute SQL inside container
docker exec -i $DB_CONTAINER mariadb -uroot -p$ROOT_PASS -e "$SQL"

echo "Database 'customers' and tables created successfully."
```

### Make it executable:
```bash
chmod +x create_customers_db.sh
```

### Run it:
```bash
./create_customers_db.sh
```
