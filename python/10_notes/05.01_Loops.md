### for

* `for` loops execute the same code as much as needed (iterating over a sequence)
* range - have 2 values `start` value and `end` value.
   * if `start` value is empty or not exists, the start value will be zero (0) by default.
  
###### 📄 template

```python
for i in range(start, end):
    code
```

###### ✏️ example: print range of values for `1` to `2`

```python
for i in range(1, 3):
    print(i);
```


###### 🖥️ output: 

```
1
2
```

###### ✏️ example: `for` loop without `start` value will be zero (0) value by default.


```python
for i in range(3):
    print(i);
```

###### 🖥️ output: 

```
0
1
2
```


###### ✏️ example: for loop

```python
sum_numbers = 0;
for i in range(1, 3):
    sum_numbers += i;

print(f"output ==> {sum_numbers}");
```


###### 🖥️ output: 

```
output ==> 3
```

-----

###### ✏️ example: for loop

```python
for i in range(3,28):
    print(f"-[{i}]-");
```

###### 🖥️ output: 

```
output ==> 3
```

-----

###### ✏️ example: count eve numbers range(10-51)

```python

for i in range(10, 51):
    if i % 2 == 0:
        count_even+=1;

# Don't change the line below

```

### while

*  `while` loop is iterating as long as a condition is `true`.

###### 📄 template

```python
while condition:
    code
```

###### ✏️ example:  Find the smallest power of 2 that is greater than a given number

```python
number = 27
power_of_two = 1

while power_of_two <= number:
    power_of_two *= 2

print(power_of_two)
```


###### 🖥️ output: 

```
32
```

###### ✏️ example: divid a number by 2, while number >= 3.5

```python
print("enter a number:");
number = 0;
number = input();
number = float(number);

while number >= 3.5:
     number = number / 2;
     print(number);
```


###### 🖥️ output: 
```python
enter a number:
44

22.0
11.0
5.5
2.75
```

###### ✏️ example:  get a number(float) divide the number by 3 as long as number >= 2.5 and print the result

```python
print("enter a number:");
number = float(input());

while number >= 2.5:
    number /= 3;
print (f"{number}");
```

###### 🖥️ output: 

```
enter a number:
67
2.4814814814814814
```

### `break` statement

* The `break` statement stops the loop. 

###### 📄 template

```python
for ...
    if i == x:
        break
```

###### ✏️ example 1: when i == 3 exit the loop (break)

```python
for i in range(8): 
    if i == 3:
        break
    print(i)
```


###### 🖥️ output: 

```
0
1
2
```

----

###### ✏️ example: 2:

* take 2 numbers
* first loop, Find and prints the **first even number greater than 5**
* Second loop, Find and prints the **first even number that divided by 7**

```python
print("Enter Min: "); 
min = int(input());
print("Enter Max: "); 
max = int(input());

for number in range (min,max):
    if number > 5 and number % 2 == 0:
        print(f"First even number greater than 5: {number}");
        break;

for number in range (min,max):
    if number % 7 == 0:
        print(f"First number divisible by 7: {number}");
        break;
```


###### 🖥️ output: 

```
Enter Min: 
2
Enter Max: 
12
First even number greater than 5: 6
First number divisible by 7: 7
```

### `continue` statement

* The `continue` statement skip to the next iteration of the loop.

###### 📄 template

```python
for ...
    if i == x:
        continue
```

###### ✏️ example 1: `continue` if number equal to 2 

```python
for number in range(1, 5):
	if number == 2:
		continue;
	print(number);
```


###### 🖥️ output: 

```
1
3
4
```

### `Range` Function

* The `range()` function generate a sequence of numbers.
* The `range()` function is been used inside loop (for() loop).

###### 📄 template

```python
range(max):
range(min, max)
range(min, max, step)
```

###### ✏️ example 1:  range 0-4 (step +1)
```
for i in range(5):
    print(i)
```

###### 🖥️ output: 

```
0 1 2 3 4
```

----

###### ✏️ example 2: range 4-5 (step +1)
```
for i in range(4, 6):
    print(i)
```

###### 🖥️ output:  

```
4 5
```

----

###### ✏️ example 3: range 1-9 (step +2)
```
for i in range(1, 10, 2):
    print(i)
```

###### 🖥️ output:  

```
1 3 5 7 9
```

----

###### ✏️ example 4: range 5-1 (step -1)
```
for i in range(5, 0, -1):
    print(i)
```

###### 🖥️ output:  

```
5 4 3 2 1
```

----

###### ✏️ example 5: 
```
for i in range(9, 0, -3):
    print(i)
```

###### 🖥️ output:  

```
9 6 3
```
