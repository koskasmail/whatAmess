### for

* loops execute the same code as much as needed.
* range -
* 
  
###### template

* loop `start` value 
* loop `end` value 

###### Note if `start` value is  empty or not exists, the start value will be zero (0) as default.
```python
for i in range(start, end):
    code
```


#### example

```python
for i in range(1, 3):
    print(i);
```


###### output: 

```
1
2
```

#### example: for without `start` value

```python
for i in range(3):
    print(i);
```

###### output: 

```
0
1
2
```


#### example: for loop


```python
sum_numbers = 0;
for i in range(1, 3):
    sum_numbers += i;

print(f"output ==> {sum_numbers}");
```


###### output: 

```
output ==> 3
```
