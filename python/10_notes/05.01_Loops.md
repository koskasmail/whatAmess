### for

* loops execute the same code as much as needed.
* range - 
  
###### template

* loop `start` value
* loop `end` value 

```python
for i in range(start, end):
    code
```


#### example

```python
for i in range(1, 3):
    print(i);
```


###### output: 

```
1
2
```


####

```python
sum_numbers = 0;
for i in range(1, 3):
    sum_numbers += i;

print(f"output ==> {sum_numbers}");
```


###### output: 

```
output ==> 3
```
