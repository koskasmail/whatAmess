# 03.04_Nested_If-Else


#### temperature check

```
temperature = 15;
weather = "unset";

if temperature <= 25:
    if temperature < 0:
        weather = "Freezing";
    elif temperature >= 0 and temperature <= 15:
        weather = "Cold";
    elif temperature >= 16 and temperature <= 25:
        weather = "Mild";
elif temperature >= 26:
    weather = "Hot";
else:
    weather = "other";
  
print(f"weather is:  {weather}");
```

###### output: 

```
weather is:  Cold
```
