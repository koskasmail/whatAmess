### Cast

* `Cast` convert the input to a different type.

###### âœï¸ example: String to number:
```python
txt = input(); #String
num = int(txt); #number
print(f"num ==> {num}");

num += 2; #number + 2
print(f"num +2 ==> {num}");
````

#### `Cast`

| Cast | Explanation | 
| :-: | :-: | 
| int() | Convert to a whole number | 
| float() | Convert to a real number | 
| bool() | Convert to a boolean | 
| str() | Convert to a string | 

###### âœï¸ example: `Cast` to float
