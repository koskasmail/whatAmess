### print

* `print()` function prints message to the `screen` or other output `device`.
* print text using "" or '' 
  
###### template

```
print ("XXX");
print (156);
```

#### example
```
stVar = "var Text";
intVar = 33;
print ("text");
print (22);
print(f" stVar ==> {stVar} and intVar ==> {intVar}");
```

###### output: 

```
text
22
stVar ==> var Text and intVar ==> 33
```
