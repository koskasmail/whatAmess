----

###### ✏️ example 1: Outer --> Inner

```python
# Outer loop
for a in range (0,2):
    print (f"{a} --> Outer loop");
    # Inner loop
    for b in range (10,12):
        print (f"   {b} --> Inner loop");
```


###### 🖥️ output: 

```
0 --> Outer loop
   10 --> Inner loop
   11 --> Inner loop
1 --> Outer loop
   10 --> Inner loop
   11 --> Inner loop
```

----

###### ✏️ example 2: print star triangle 4x4

```python
# print triangle 4,4 stars
for row in range(1,5):  # 2 rows
    line = ""
    for col in range(0, row):  # 3 columns
        line += "*"
    print(line)
```


###### 🖥️ output: 

```
*
**
***
****
```

###### ✏️ example 3: user input to print `*`.

```python
print("Enter Rows value: ");
rows = int(input());

print("Enter Cols value: ");
cols = int(input());

for a in range (rows):
    print("");
    for b in range (cols):
        print("*", end="");
```


###### ✏️ example 5: xxx


* Write a program that gets a dynamic number of input values.
* The first input tells you how many numbers will follow.
* The next inputs are whole numbers that you need to sum.
* In the end, print the sum of all the input numbers (not including the first input).
* .
* Example walkthrough:
* 3
   * 1
   * 5
   * 6

* Explanation:
   * Two set of input values
   * input #1 is the value `3` and pressent the total number to come. 
   * input #2 is set of numbers the user entered (3 in this case) --> 1, 5, 6
      * output: sum all input 2 number and print result ---> 1 + 5 + 6 = 12
      * print: output result
* TIP: random number can be user typing numbers.


###### ✏️ code
```python
print("Enter a number:");
count = int(input()) ;

total = 0;

for x in range(count):
    print(f"enter a number {x+1}: ");
    total += int(input());

print(f"total = {total}");
```

###### ✍️ user input: 

```
Enter a number:
4
enter a number 1: 
1
enter a number 2: 
2
enter a number 3: 
3
enter a number 4: 
4
```

###### 🖥️ output: 

```
total = 10
```
