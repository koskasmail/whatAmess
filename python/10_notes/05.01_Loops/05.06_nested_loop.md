----

###### ✏️ example 1: Outer --> Inner

```python
# Outer loop
for a in range (0,2):
    print (f"{a} --> Outer loop");
    # Inner loop
    for b in range (10,12):
        print (f"   {b} --> Inner loop");
```


###### 🖥️ output: 

```
0 --> Outer loop
   10 --> Inner loop
   11 --> Inner loop
1 --> Outer loop
   10 --> Inner loop
   11 --> Inner loop
```

----

###### ✏️ example 2: print star triangle 4x4

```python
# print triangle 4,4 stars
for row in range(1,5):  # 2 rows
    line = ""
    for col in range(0, row):  # 3 columns
        line += "*"
    print(line)
```


###### 🖥️ output: 

```
*
**
***
****
```

----

###### ✏️ example 3: user input to print `*`.

```python
print("Enter Rows value: ");
rows = int(input());

print("Enter Cols value: ");
cols = int(input());

for a in range (rows):
    print("");
    for b in range (cols):
        print("*", end="");
```

###### ✍️ user input: 

```
Enter Rows value: 
3
Enter Cols value: 
5
```

###### 🖥️ output: 

```
*****
*****
*****
```
