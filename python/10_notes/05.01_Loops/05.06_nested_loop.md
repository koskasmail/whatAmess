###### ✏️ example 1: Outer --> Inner

```python
# Outer loop
for a in range (0,2):
    print (f"{a} --> Outer loop");
    # Inner loop
    for b in range (10,12):

```

```

   10 --> Inner loop
   11 --> Inner loop
1 --> Outer loop

```

###### ✏️ example 5: xxx

* Write a program that finds all pairs of numbers that multiply to give n using numbers from 1 to n (inclusive).
* The program should show all possible combinations, including duplicate pairs in reverse order.
* For example, both "1 6" and "6 1" should be shown, as they are considered different arrangements of the same pair.
* Numbers can also be paired with themselves if their product equals n.
* .
* Important: Only consider numbers in the range from 1 to n. If n is less than 1, no pairs exist.

###### sample:

```
For example if n = 12, the output should be:
1 12
2 6
3 4
4 3
6 2
12 1

Because:
1 * 12 = 12
2 * 6 = 12
3 * 4 = 12
4 * 3 = 12
6 * 2 = 12
12 * 1 = 12

Tip:
Your job is to find all pairs of numbers (two numbers together) that:
are between 1 and n
when you multiply them, the result is exactly n
and you must show both directions  
(for example: 2 6 and also 6 2)
```

###### ✏️ code

```python
print("Enter n: ");
n = int(input());

if n < 1:
    print("No pairs exist.");
else:
    for a in range(1, n + 1):
        for b in range(1, n + 1):
            if a * b == n:
                print(a, b);
```

###### ✍️ user input: 

```
22
```

###### 🖥️ output: 

```
1 22
2 11
11 2
22 1
```

###### ✏️ example 6: dynamic number of input values

* Write a program that gets a dynamic number of input values.
* The first input tells you how many numbers will follow.
* The next inputs are whole numbers that you need to sum.
* In the end, print the sum of all the input numbers (not including the first input).
* .
* Example walkthrough:
* 3
   * 1
   * 5
   * 6

* Explanation:
   * Two set of input values
   * input #1 is the value `3` and pressent the total number to come. 
   * input #2 is set of numbers the user entered (3 in this case) --> 1, 5, 6
      * output: sum all input 2 number and print result ---> 1 + 5 + 6 = 12
      * print: output result
* TIP: random number can be user typing numbers.


###### ✏️ code
```python
print("Enter a number:");
count = int(input()) ;

total = 0;

for x in range(count):
    print(f"enter a number {x+1}: ");
    total += int(input());

print(f"total = {total}");
```

###### ✍️ user input: 

```
Enter a number:
4
enter a number 1: 
1
enter a number 2: 
2
enter a number 3: 
3
enter a number 4: 
4
```

###### 🖥️ output: 

```
total = 10
```
