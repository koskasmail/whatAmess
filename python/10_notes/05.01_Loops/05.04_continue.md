# 05.04_continue

#### `continue` 

```python
for i in range(start, end):
    code
```

----

###### ✏️ example 1: `continue` if number is even.

```python
for number in range(1, 5):
	if number == 2:
		continue;
	print(number);
```


###### 🖥️ output: 

```
1
3
4
```

----

###### ✏️ example 2:  `continue` if number is odd.

###### 2.1: result fraction not equal to zero `i % 2 != 0` 
```python
for i in range(1, 12):
    if i % 2 != 0:
        print(i)
```

###### 2.2: result fraction bigger than zero `i % 2 > 0` 
```
print("print odd number");
for i in range(1, 12):
    if i % 2 > 0:
        print(i)
```

###### 🖥️ output: 

```
1
3
5
7
9
11
```
