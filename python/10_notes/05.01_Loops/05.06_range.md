# 05.06_range

### `Range` Function

* The `range()` function generate a sequence of numbers.
* The `range()` function is been used inside loop (for() loop).

###### 📄 template

```python
range(max):
range(min, max)
range(min, max, step)
```

###### ✏️ example 1:  range 0-4 (step +1)
```python
for i in range(5):
    print(i)
```

###### 🖥️ output: 

```
0 1 2 3 4
```

----

###### ✏️ example 2: range 4-5 (step +1)
```python
for i in range(4, 6):
    print(i)
```

###### 🖥️ output:  

```
4 5
```

----

###### ✏️ example 3: range 1-9 (step +2)
```python
for i in range(1, 10, 2):
    print(i)
```

###### 🖥️ output:  

```
1 3 5 7 9
```

----

###### ✏️ example 4: range 5-1 (step -1)
```python
for i in range(5, 0, -1):
    print(i)
```

###### 🖥️ output:  

```
5 4 3 2 1
```

----

###### ✏️ example 5: 
```python
for i in range(9, 0, -3):
    print(i)
```

###### 🖥️ output:


###### ✏️ example 6: user enter: start,stop,step values.

```python
# Get input from user
print("Enter START value: ");
start = int(input());

print("Enter STOP value: ");
end = int(input());

print("Enter STEP value: ");
step = int(input());

# range - print loop.
for i in range(start, end, step):
    print(i)
```


###### 🖥️ user input: 

```
Enter START value: 
2
Enter STOP value: 
10
Enter STEP value: 
3
```

###### 🖥️ output:  

```
2 5 8
```

###### ✏️ example 7: BUG - python do nothing

```python
for i in range(5, 10, -1):
    print(i)
```

###### 🖥️ output:  

* START value: 5
* STOP value: 10
* STEP value: -1

###### TIP
* The loop produces no output because your range(5, 10, -1) is an empty range.
* But you cannot count downward from 5 and reach 10, so Python stops immediately and the loop never runs.
* if the program was able to execute the output: 5 4 3 2 1 0 -1.....N

###### ✏️ example 9: Print First 8 odd numbers from 15

```python
print("\nFirst 8 odd numbers from 15:")
# Your code here
count = 0;
number = 15;
while count < 8:
    if number % 2 == 1:
        count = count + 1;
        print(number, end=", ");
    number = number + 1;
```

###### 🖥️ output:  

```
15, 17, 19, 21, 23, 25, 27, 29,
```

###### ✏️ example 10: Counting backwards and print number divisible by 5

###### ✏️ example 11: Product of numbers divisible by 3 (1-30) - print only the number, not "Product = number"

```python

        product *= i;
print(product);


```

