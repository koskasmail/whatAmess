# 05.06_range

### `Range` Function

* The `range()` function generate a sequence of numbers.
* The `range()` function is been used inside loop (for() loop).

###### 📄 template

```python
range(max):
range(min, max)
range(min, max, step)
```

###### ✏️ example 1:  range 0-4 (step +1)
```
for i in range(5):
    print(i)
```

###### 🖥️ output: 

```
0 1 2 3 4
```

----

###### ✏️ example 2: range 4-5 (step +1)
```
for i in range(4, 6):
    print(i)
```

###### 🖥️ output:  

```
4 5
```

----

###### ✏️ example 3: range 1-9 (step +2)
```
for i in range(1, 10, 2):
    print(i)
```

###### 🖥️ output:  

```
1 3 5 7 9
```

----

###### ✏️ example 4: range 5-1 (step -1)
```
for i in range(5, 0, -1):
    print(i)
```

###### 🖥️ output:  

```
5 4 3 2 1
```

----

###### ✏️ example 5: 
```
for i in range(9, 0, -3):
    print(i)
```

###### 🖥️ output:  

```
9 6 3
```
