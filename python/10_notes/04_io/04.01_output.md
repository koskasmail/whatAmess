# 04.01_output


#### `print()` function

```
print("xxx");
print(22);
print(f"{var}");
```

#### print text

```
age = 10
print("His age is: age")
```

###### output
```
His age is: age
```

----

#### print text and variable value

```
print(f"variable ==> {variable}")
```

```
age = 10
print(f"His age is: {age}");
```

###### output
```
His age is: 10
```

----


#### print() variable value

```
num = 33;
print(f"num: {num}");
```

###### 🖥️ output:  

```
num: 33
```

----

###### ✏️ example: print `end parameter` end="--"
```python
print("Hello", end="--");
print("World");
```

###### 🖥️ output:  

```
Hello--World
```

----

###### ✏️ example: print `end parameter` end="..."
```python
print("Loading", end="...");
```

###### 🖥️ output:  

```
Loading...
```

----

###### ✏️ example: print `end parameter` end="+"
```python
print("A", end="+");
print("B");
```

###### 🖥️ output:  

```
A+B
```

----
