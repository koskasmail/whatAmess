# 05.01_Bill_Split_Calculator

```py

###---[welcome message]----###
print("Bill Split Calculator");

###---[get user input]----###
print("Enter bill amount");
bill = input();

print("tip percentage");
tip = input();

print("How Many People are sharing bill");
totalPerson = input();

###---[convert to float]----###
bill = float(bill);
tip = float(tip);
totalPerson = int(totalPerson);

###---[tip amount (% to $)]----###
tip_amount = (tip / 100) * bill;
print(f"total =  {tip}");

###---[totla pay (bill + tip)]----###
total = tip_amount + bill;
print(f"total (bill + tip) =  {total}");

amountPerPerson = total / totalPerson;
print(f"{amountPerPerson}");
```
