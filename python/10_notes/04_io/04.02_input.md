# 04.02_input


#### `input()` function

```
var = input()
```

#### example: get user name

```
name = input(); # Don't change this line

print(f"Hello, {name}");
```

###### output
```
Hello, Bob
```

----

#### example: user's name and age, calculates and prints how old they will be in 10 years.

```
# Get user input
print("Enter Name: ")
name = input()

print("Enter Age: ")
age = int(input())

# Convert age to integer and add 10
future_age = int(age) + 10

# Print the result
print(f"In 10 years, {name} will be {future_age} years old.")
```

###### output
```
In 10 years, Sam will be 5 years old.
```
