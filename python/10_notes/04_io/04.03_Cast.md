#### example: age print

* task:
   * gets the user's age as input.
   * print the number of missing years till 120
      * format like if user pressed 25
         * "95 years till 120".

```python
print("Enter age: ") 
age = 0
age = input()
age = int(age)
left = 120 - age;
print(f"{left} years till 120")
```
