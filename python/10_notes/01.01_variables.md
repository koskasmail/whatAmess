#### string

###### `char` 
* `char` is a single character
* ‚úèÔ∏è example:: 1, 2, 3,..., a, b..., D, E,..., !, @, #,..)

###### ‚úèÔ∏è example: print 'string/text'

###### üñ•Ô∏è output:

###### `str` (string) 
* `str` (string) is a multiple chars.
* To initialize a string value in a variable, enclose it within `single` quote or `double` quote marks:

###### ‚úèÔ∏è example:
```python

```

###### üñ•Ô∏è output: 

```python

s2 = double quotation string
```

#### Naming_Conventions

* `Naming Conventions` is a convention of naming things (variable naming, etc...)
* python Naming Conventions is `snake case`.
    * example:
       * is_free = True;
       * age = 25;
       * greeting = 'welcome';

###### ‚úèÔ∏è example:
```python
is_free = True;
age = 25;
greeting = 'welcome';

print(f'is_free = {is_free}');
print(f'age = {age}');
print(f'greeting = {greeting}');
```

###### üñ•Ô∏è output: 

```
is_free = True
age = 25
greeting = welcome
```
