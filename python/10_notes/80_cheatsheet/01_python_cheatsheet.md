# 01_python_cheatsheet



#### Arithmetic_Operators

| Operator | Description | Example | 
| :-: | :-: |  :-: | 
| + | Addition | 1 + 2 = 3  |
| - | Subtraction | 3 - 2 = 1  |
| * | Multiplication | 3 * 3 = 9 |
| / | Division | 6 / 3 = 2  |
| % | Modulus | 5 % 2 = 1 |
| // | 	Floor division | 5 // 2 = 2 |
| ** | Exponentiation | 3 ** 2 = 9 |

----

#### Arithmetic_Shortcuts 

| Operator | Shortcut | Example | 
| :-: | :-: |  :-: | 
| + | += | a += 2 |
| - | -= | b -= 3 |
| * | *= | c *= 4 |
| / | /= | d /= 5 |
| % | %= | e %= 6 |
| **= | **= | f **= 3 |
| // | //= | g //= 2 |
