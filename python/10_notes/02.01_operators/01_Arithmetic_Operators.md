# 01_Arithmetic_Operators


#### Arithmetic_Operators

| Operator | Description | Example | 
| :-: | :-: |  :-: | 
| + | Addition | 1 + 2 = 3  |
| - | Subtraction | 3 - 2 = 1  |
| * | Multiplication | 3 * 3 = 9 |
| / | Division | 6 / 3 = 2  |
| % | Modulus | 5 % 2 = 1 |
| // | 	Floor division | 5 // 2 = 2 |
| ** | Exponentiation | 3 ** 2 = 9 |

----

#### ex2

```
count = 10;

count += 4;
count *= 2;
count -= 1;
count /= 1;

print(f"count = {count}")
```

###### output
```
count = 7
```
