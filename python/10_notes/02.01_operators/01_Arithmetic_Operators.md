# 01_Arithmetic_Operators

#### Arithmetic_Operators

| Operator | Description | Example | explanation | 
| :-: | :-: | :-: | :-: |
| + | Addition | 1 + 2 = 3  | Adds two numbers |
| - | Subtraction | 3 - 2 = 1  | Subtracts one number from another |
| * | Multiplication | 3 * 3 = 9 | Multiplies two numbers |
| / | Division | 6 / 3 = 2  | Divides one number by another and returns a float |
| ** | Exponentiation | 3 ** 2 = 9 | exponentiation, raises a number to the power of another number |
| // | Floor division | 5 // 2 = 2 | divides 2 numbers and then rounds the result down to the nearest whole number |
| % | Modulus Modulo | 5 % 2 = 1 | It returns the remainder/fraction after dividing one number by another. |


----

#### `+` Addition
```
number = 2;

number = number + 2;  ## =4
print (number);
```

###### output
```
4
```

----

#### `-` Subtraction
```
number = 4;
number = number - 1;  ## =3
print (number);
```

###### output
```
3
```

----

#### `*` Multiplication
```
number = 3;
number = number * 4; ## =12
print (number);
```

###### output
```
12
```

----

#### `/` Division 
```
number = 12;
number = number / 2; ## =6.0
print (number);
```

###### output
```
6.0
```

----

#### `**` exponentiation (power of) 

```
number = 6;
number = number ** 2; ## =36.0
print (number);
```

###### output

```
36
```

----

#### `//` Floor division
```
number = 36;
number = number // 2; ## =18
print (number);
```

###### output
```
18
```

----

#### `%` Modulus/Modulo
```
number = 18;
number = number % 2; ## =0.0
print (number);
```

###### output
```
0.0
```
