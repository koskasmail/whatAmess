# 03_Arithmetic_Shortcuts 


#### Arithmetic_Shortcuts 

| Operator | Shortcut | Operator | Shortcut|
| :-: | :-: |  :-: |  :-: | 
| + | += | a = a + 2 | a += 2 |
| - | -= | b = b - 3 | b -= 3 |
| * | *= | c = c * 4 | c *= 4 |
| / | /= | d = d / 5 | d /= 5 |
| % | %= | e = e % 6 | e %= 6 |
| **= | **= | f = f ** 3 | f **= 3 |
| // | //= | g = g // 2 | g //= 2

----


#### ex2

```
count = 10;

count += 4;
count *= 2;
count -= 1;
count /= 1;

print(f"count = {count}")
```

###### output
```
count = 7
```
