If you want **Python to print your disk size** (total, used, free), you can do it easily without installing anything extra.

Here are two clean approaches.

---

# ðŸŸ¦ **Method 1: Using `shutil` (builtâ€‘in, simplest)**

```python
import shutil

total, used, free = shutil.disk_usage("/")

print("Total:", total // (1024**3), "GB")
print("Used:", used // (1024**3), "GB")
print("Free:", free // (1024**3), "GB")
```

This prints the disk size of your root filesystem `/`.

---

# ðŸŸ© **Method 2: Pretty formatted output**

```python
import shutil

total, used, free = shutil.disk_usage("/")

print(f"Disk Size:")
print(f"  Total: {total / (1024**3):.2f} GB")
print(f"  Used : {used / (1024**3):.2f} GB")
print(f"  Free : {free / (1024**3):.2f} GB")
```

# ðŸŸ§ Want to list all mounted disks?

```python
import os
import shutil

mounts = ["/", "/home", "/boot", "/mnt"]

for m in mounts:
    if os.path.ismount(m):
        total, used, free = shutil.disk_usage(m)
        print(f"{m}: {total / (1024**3):.2f} GB total")
```
