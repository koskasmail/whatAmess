
### nintendo nes (nis) programming environment.

#### Herramientas necesarias 
* editor
   * vscode
* ensamblador
* emulador

#### vscode
* install vscode
* addons
   *  ca65 Macro Assembler Language Support (6502/65816)


#### assembler
* convert plain text assembly code into binary machine code.
   * <.asm> into <.nes>



#### 01.sample.s

``` 
lda $01
sta $02
```

* save file extensions: <.s>
   *  <.s> is 6502 assembly
