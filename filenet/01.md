**Quick Answer:** IBM FileNet provides APIs to **create (write)** and **retrieve (read)** documents from its Content Engine. You typically use the `Document` class to create a new document, set properties, and add content, and then use `fetchInstance` or `getInstance` to retrieve documents.  

---

## üìò IBM FileNet Cheat Sheet: Reading & Writing Documents

### üîë Key Concepts
- **Object Store**: Repository where documents are stored.
- **Document Object**: Represents a single version of a document.
- **Content Element**: The actual file (Word, PDF, image, etc.) attached to the document.
- **Properties**: Metadata fields (title, author, custom attributes).

---

### üìù Writing (Creating) a Document
Steps:
1. Connect to the **Content Engine**.
2. Get a reference to the **Object Store**.
3. Create a new **Document** object.
4. Set **properties** (e.g., name, class).
5. Add **content** (binary stream).
6. Save the document.

**Java Example:**
```java
// Connect to Object Store
ObjectStore os = Factory.ObjectStore.fetchInstance(domain, "MyObjectStore", null);

// Create a new Document
Document doc = Factory.Document.createInstance(os, "Document");

// Set properties
doc.getProperties().putValue("DocumentTitle", "SampleDoc");

// Add content
ContentElementList cel = Factory.ContentElement.createList();
ContentTransfer ct = Factory.ContentTransfer.createInstance();
ct.setCaptureSource(new FileInputStream("C:\\sample.pdf"));
cel.add(ct);
doc.set_ContentElements(cel);

// Save document
doc.checkin(AutoClassify.DO_NOT_AUTO_CLASSIFY, CheckinType.MAJOR_VERSION);
doc.save(RefreshMode.REFRESH);
```

---

### üìñ Reading (Retrieving) a Document
Steps:
1. Fetch the document by **ID** or **path**.
2. Access its **properties**.
3. Retrieve its **content stream**.

**Java Example:**
```java
// Fetch document by ID
Id docId = new Id("{YOUR-DOCUMENT-ID}");
Document doc = Factory.Document.fetchInstance(os, docId, null);

// Read properties
String title = doc.getProperties().getStringValue("DocumentTitle");

// Get content
ContentElementList cel = doc.get_ContentElements();
ContentTransfer ct = (ContentTransfer) cel.get(0);
InputStream is = ct.accessContentStream();

// Example: Save to local file
Files.copy(is, Paths.get("C:\\downloaded.pdf"));
```

---

### ‚ö° Common Operations
- **Update properties**: `doc.getProperties().putValue("Author", "Yaron");`
- **Delete document**: `doc.delete(); doc.save(RefreshMode.NO_REFRESH);`
- **Search documents**: Use **SQL-like queries** with FileNet APIs.

---

### üö® Tips & Gotchas
- Always **check in** documents after adding content.
- Use **major/minor versions** to manage document lifecycle.
- Ensure **security permissions** are set correctly when saving.
