ניתן להשתמש בבודק האיות המובנה (Spell Checker) של Eclipse כדי לזהות שגיאות בטקסט ובערות קוד. [1, 2] 
הפעלה והגדרות בסיסיות
כדי להפעיל את בודק האיות, בצע את השלבים הבאים: [1, 2] 

   1. עבור לתפריט Window ובחר ב-Preferences.
   2. בצד שמאל, נווט לנתיב: General > Editors > Text Editors > Spelling.
   3. סמן את תיבת הסימון Enable spell checking.
   4. כאן תוכל להגדיר אפשרויות נוספות, כמו התעלמות ממילים הכוללות ספרות, אותיות רישיות או כתובות אינטרנט. [2, 3, 4, 5, 6] 

שימוש בתוך העורך
לאחר הפעלת התכונה, Eclipse יסמן מילים שגויות בקו תחתון אדום. [1, 7, 8] 

* תיקון מהיר: העמד את הסמן על המילה המסומנת ולחץ על Ctrl + 1 (או העבר את העכבר מעל המילה) כדי לראות רשימת הצעות לתיקון.
* הוספה למילון: אם המילה נכונה אך לא מוכרת, ניתן לבחור ב-Add to dictionary מתוך תפריט התיקון המהיר.
* בדיקת איות ידנית: בחלק מסוגי הקבצים (כמו HTML או XML), ניתן להפעיל את הבדיקה דרך Edit > Spell Check. [5, 6, 9, 10] 

ניהול מילון משתמש
ניתן להוסיף מילון אישי כדי למנוע סימון של מונחים טכניים או שמות: [6, 11] 

* צור קובץ טקסט פשוט (.txt) שבו כל מילה בשורה נפרדת.
* בהגדרות ה-Spelling (בשלב הראשון לעיל), תחת User defined dictionary, לחץ על Browse ובחר את הקובץ שיצרת. [6, 12]
