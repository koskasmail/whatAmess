### Overview

Below is a **pure Java desktop app (Swing)** that runs on **Windows and Ubuntu**, reads a JSON file periodically, and shows a **live-updating graph** of your metrics—no OS commands, no platform-specific tricks.

It uses:

- **Swing** for UI  
- **Jackson** for JSON parsing  
- **JFreeChart** for charts   [ZetCode](https://zetcode.com/java/jfreechart/)  [Baeldung](https://www.baeldung.com/jfreechart-visualize-data)  

You can swap libraries if you prefer, but this is a clean, common setup.

---

### Example JSON format

Assume your JSON file (e.g. `metrics.json`) looks like this and is updated by some other process:

```json
{
  "timestamp": 1717320000000,
  "diskSize": 512,
  "memorySize": 16384,
  "networkSpeed": 120,
  "diskSpeed": 300,
  "cpuUsage": 45
}
```

You can add/remove fields as needed—just keep the names in sync with the code.

### Full Java example: live graph from JSON file

```java
import com.fasterxml.jackson.databind.JsonNode;
import com.fasterxml.jackson.databind.ObjectMapper;
import org.jfree.chart.ChartFactory;
import org.jfree.chart.ChartPanel;
import org.jfree.chart.JFreeChart;
import org.jfree.data.time.Millisecond;
import org.jfree.data.time.TimeSeries;
import org.jfree.data.time.TimeSeriesCollection;

import javax.swing.*;
import java.awt.*;
import java.io.File;
import java.io.IOException;
import java.nio.file.Files;
import java.nio.file.Paths;

public class LiveMetricsViewer {

    private static final String JSON_PATH = "metrics.json";

    private final ObjectMapper mapper = new ObjectMapper();

    // Time series for each metric
    private final TimeSeries diskSizeSeries = new TimeSeries("Disk Size");
    private final TimeSeries memorySizeSeries = new TimeSeries("Memory Size");
    private final TimeSeries networkSpeedSeries = new TimeSeries("Network Speed");
    private final TimeSeries diskSpeedSeries = new TimeSeries("Disk Speed");

    public static void main(String[] args) {
        SwingUtilities.invokeLater(() -> {
            LiveMetricsViewer viewer = new LiveMetricsViewer();
            viewer.createAndShowUI();
            viewer.startPolling();
        });
    }

    private void createAndShowUI() {
        JFrame frame = new JFrame("Live Metrics Graph");
        frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        frame.setLayout(new GridLayout(2, 2));

        // Create charts
        frame.add(createChartPanel("Disk Size", "Time", "GB", diskSizeSeries));
        frame.add(createChartPanel("Memory Size", "Time", "MB", memorySizeSeries));
        frame.add(createChartPanel("Network Speed", "Time", "Mbps", networkSpeedSeries));
        frame.add(createChartPanel("Disk Speed", "Time", "MB/s", diskSpeedSeries));

        frame.setSize(1000, 600);
        frame.setLocationRelativeTo(null);
        frame.setVisible(true);
    }

    private ChartPanel createChartPanel(String title, String xLabel, String yLabel, TimeSeries series) {
        TimeSeriesCollection dataset = new TimeSeriesCollection(series);
        JFreeChart chart = ChartFactory.createTimeSeriesChart(
                title, xLabel, yLabel, dataset,
                false, true, false
        );
        return new ChartPanel(chart);
    }

    private void startPolling() {
        // Pure Java timer (no OS commands), runs every 1 second
        int delayMs = 1000;
        new Timer(delayMs, e -> readAndUpdate()).start();
    }

    private void readAndUpdate() {
        try {
            if (!Files.exists(Paths.get(JSON_PATH))) {
                return;
            }

            JsonNode root = mapper.readTree(new File(JSON_PATH));

            // If your JSON has its own timestamp, you can use it.
            // Here we just use "now" for the time axis.
            Millisecond now = new Millisecond();

            double diskSize = getDouble(root, "diskSize");
            double memorySize = getDouble(root, "memorySize");
            double networkSpeed = getDouble(root, "networkSpeed");
            double diskSpeed = getDouble(root, "diskSpeed");

            diskSizeSeries.addOrUpdate(now, diskSize);
            memorySizeSeries.addOrUpdate(now, memorySize);
            networkSpeedSeries.addOrUpdate(now, networkSpeed);
            diskSpeedSeries.addOrUpdate(now, diskSpeed);

        } catch (IOException ex) {
            ex.printStackTrace();
        }
    }

    private double getDouble(JsonNode root, String field) {
        JsonNode node = root.get(field);
        if (node == null || !node.isNumber()) {
            return Double.NaN;
        }
        return node.asDouble();
    }
}
```

### How to use

1. **Build** with Maven (or add the jars manually).
2. **Run** `LiveMetricsViewer`.
3. Ensure `metrics.json` is in the **working directory** and is updated periodically by your own process.
4. The graphs will update every second with the latest values.
