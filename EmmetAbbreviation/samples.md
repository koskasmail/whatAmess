
# Emmet Abbreviation

### `div` and 4 sub `div`
```
div>div*4
```
##### example:
```
<div>
    <div></div>
    <div></div>
    <div></div>
    <div></div>
</div>
```

### ul x3
```
ul*3
```
##### result:
```
<ul></ul>
<ul></ul>
<ul></ul>
```
