## ğŸ”¤ String <a name="string"></a>

### `char`
- A **char** is a single character.  
  Examples: `a`, `D`, `3`, `!`, `@`

```python
char_c = 'b'
char_d = "D"

print(f"char_c = {char_c}")
print(f"char_d = {char_d}")
```

**Output:**
```
char_c = b
char_d = D
```

### `str` (string)
- A **string** is a sequence of characters.
- Use single `' '` or double `" "` quotes.

```python
s1 = 'single quotation string'
s2 = "double quotation string"

print(f"s1 = {s1}")
print(f"s2 = {s2}")
```

**Output:**
```
s1 = single quotation string
s2 = double quotation string
```

## ğŸ Naming Conventions <a name="Naming_Conventions"></a>

- Python uses **snake_case** for variable names.
- Examples:
  - `is_free = True`
  - `age = 25`
  - `greeting = 'welcome'`

```python

print(f"age = {age}")
print(f"greeting = {greeting}")
```

**Output:**
```
is_free = True
age = 25
greeting = welcome
```
